# StudySnap AI v2

Cleaner MVP with:
- drag/drop upload
- screenshot paste support
- file preview
- timeout/error handling
- modes: Teach, Hints, Check, Study Guide
- recent sessions saved locally
- much better UI

## Run

```bash
npm install
cp .env.example .env.local
npm run dev
```

Put your real OpenAI API key in `.env.local`.
